Privacy Policy

NOC Tune membantu mengukur kualitas akses ke website target. Beberapa data teknis dibaca agar hasilnya lebih akurat. Anda tetap memegang kendali penuh, terutama untuk fitur contribute yang bisa dibuat otomatis atau manual.

Data yang dibaca saat test

  • Hasil TTFB, DNS lookup, connect time, total time, dan status HTTP.
  • Informasi perangkat dan jaringan seperti nama device, OS, RSSI, SSID, band WiFi, channel, DNS, ISP, dan IP publik.
  • Koordinat lokasi jika browser geolocation diizinkan, agar hasil test mewakili lokasi sebenarnya.

Data yang disimpan lokal

  • Session info, hasil ping, hasil TTFB, CSV, dan report di folder results.
  • Config test seperti target, threshold, brand, no internet, dan pilihan auto contribute.
  • Lokasi presisi terakhir di file lokal agar export dan report tetap konsisten.

Kapan data dikirim keluar

  • Jika Auto Contribute aktif, setiap row hasil test akan langsung dikirim ke qosmic.solusee.id.
  • Jika Auto Contribute mati, data tidak dikirim otomatis dan hanya terkirim saat tombol Contribute ditekan.
  • Reverse geocoding memakai layanan OpenStreetMap untuk mengubah koordinat menjadi nama kota/region.

Kendali di tangan user

  • Anda bisa menyalakan atau mematikan Auto Contribute dari Configuration.
  • Brand dan No Internet bersifat opsional dan bisa dikosongkan.
  • Anda bisa menjalankan test hanya untuk kebutuhan lokal tanpa mengirim hasil ke layanan kontribusi.
Ringkasnya: aplikasi ini fokus pada data teknis jaringan untuk kebutuhan pengukuran. Pengiriman ke layanan QoSMic bersifat opsional dan bisa diatur dari config.

About NOC Tune

NOC Tune adalah aplikasi browser-based untuk mengukur Time To First Byte secara otomatis. Tool ini membantu melihat kualitas akses ke target tertentu dengan cepat, lengkap, dan mudah dipahami.

Apa yang dilakukan aplikasi ini

  • Mengukur TTFB ke satu atau banyak target secara berulang.
  • Mencatat detail teknis jaringan saat test berlangsung.
  • Menampilkan hasil secara real-time di browser.

Kelebihan utamanya

  • Auto ukur TTFB tanpa perlu command manual berulang.
  • Auto mencatat detail penting seperti DNS, WiFi, lokasi, ISP, dan perangkat.
  • Visualisasi hasil dalam tabel, summary card, dan chart.
  • Hasil bisa diunduh sebagai CSV dan report.

Contribute ke QoSMic

NOC Tune bisa mengirim hasil ke qosmic.solusee.id agar data pengukuran terkumpul. Fitur ini opsional. Anda bisa memilih submit otomatis per row atau submit manual setelah test selesai.

Cocok dipakai untuk

  • Validasi kualitas akses ke CDN atau website tertentu.
  • Membandingkan kondisi jaringan antar lokasi atau antar band WiFi.
  • Membuat bukti teknis yang lebih lengkap saat troubleshooting.
Singkatnya: NOC Tune membantu Anda mengukur TTFB secara otomatis, melihat konteks jaringan dengan lebih lengkap, dan membagikan hasil ke QoSMic bila memang dibutuhkan.
